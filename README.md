## دستیار هوشمند مکانی با پشتیبانی از زبان فارسی

**مپاتون** یک نرم‌افزار هوشمند مکانی است که پرس‌وجوهای طبیعی کاربران را درک می‌کند و نتیجه را به‌صورت قابل مشاهده روی نقشه نمایش می‌دهد.

با استفاده از مپاتون، کاربران بدون نیاز به آشنایی با ابزارهای تخصصی GIS می‌توانند مکان‌ها را جست‌وجو کنند، مسیر دریافت کنند و تحلیل‌های مکانی انجام دهند.

🔗 وب‌سایت پروژه: [mapathon.ir](https://mapathon.ir)

</div>

## درباره مپاتون

مپاتون با ترکیب هوش مصنوعی، داده‌های جغرافیایی و رابط کاربری ساده، پیچیدگی‌های کار با داده‌های مکانی را کاهش می‌دهد.

کاربر می‌تواند درخواست خود را به زبان طبیعی فارسی وارد کند؛ برای مثال:

```text
نزدیک‌ترین بیمارستان‌ها به موقعیت فعلی من را پیدا کن
```

یا:

```text
از میدان آزادی تا میدان ونک مسیریابی کن و داروخانه‌های موجود در مسیر را نشان بده
```

مپاتون درخواست کاربر را تحلیل کرده و پاسخ را به شکل اطلاعات مکانی، نشانگر، مسیر یا لایه‌های قابل مشاهده روی نقشه نمایش می‌دهد.

## قابلیت‌ها

- مسیریابی هوشمند
- جست‌وجوی مکان‌ها در مسیر

- نمایش مبدأ و مقصد با نشانگرهای متفاوت
- پشتیبانی از رابط کاربری راست‌به‌چپ و زبان فارسی
- استفاده از فونت فارسی Vazirmatn
- تجربه کاربری ساده و مناسب برای کاربران عمومی
- قابلیت توسعه برای کاربردهای مختلف شهری، سازمانی و تجاری

## هدف پروژه

هدف مپاتون ایجاد یک رابط ساده، عمومی و توسعه‌پذیر برای دسترسی به داده‌های مکانی است؛ به‌گونه‌ای که کاربر بتواند بدون استفاده مستقیم از ابزارهای پیچیده GIS، نیاز مکانی خود را با زبان طبیعی بیان کند.

مپاتون تلاش می‌کند فاصله میان کاربر عمومی و فناوری‌های تخصصی مکانی را کاهش دهد.

## معماری کلی

پروژه از دو بخش اصلی تشکیل شده است:

### Frontend

رابط کاربری وب برای تعامل با کاربر و نمایش نقشه و نتایج مکانی.

فناوری‌های اصلی:

- React
- TypeScript
- Vite
- MapLibre GL
- CSS
- Vazirmatn

### Backend

سرویس پردازش درخواست‌ها، تحلیل پرس‌وجوهای مکانی و ارتباط با سرویس‌های داده و نقشه.

فناوری‌های اصلی:

- Python
- FastAPI
- PostgreSQL
- PostGIS
- سرویس‌های ژئوکدینگ و مسیریابی
- مدل‌های هوش مصنوعی

## اجرای Frontend در محیط توسعه

ابتدا وارد پوشه فرانت‌اند شوید:

```bash
cd frontend
```

سپس وابستگی‌ها را نصب کنید:

```bash
npm install
```

برای اجرای پروژه:

```bash
npm run dev
```

پروژه به‌صورت پیش‌فرض در آدرس زیر در دسترس خواهد بود:

```text
http://localhost:5173
```

برای ساخت نسخه Production:

```bash
npm run build
```

برای بررسی نسخه Build‌شده:

```bash
npm run preview
```

---

## اجرای Backend در محیط توسعه

وارد پوشه بک‌اند شوید:

```bash
cd backend
```

یک محیط مجازی ایجاد و فعال کنید:

```bash
python -m venv .venv
source .venv/bin/activate
```

در ویندوز:

```powershell
.venv\Scripts\activate
```

سپس وابستگی‌ها را نصب کنید:

```bash
pip install -r requirements.txt
```

برای اجرای سرویس:

```bash
uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

سرویس Backend در آدرس زیر قابل دسترسی خواهد بود:

```text
http://localhost:8000
```

مستندات API در FastAPI:

```text
http://localhost:8000/docs
```

## متغیرهای محیطی

اطلاعات حساس نباید مستقیماً در کد یا فایل README قرار بگیرند. فایل `.env.example` را ایجاد کنید:

```env
# Frontend
VITE_API_URL=http://localhost:8000
NEXT_PUBLIC_API_URL=http://localhost:8000
NEXT_PUBLIC_MAP_STYLE_URL=

# Backend
DATABASE_URL=postgresql://USER:PASSWORD@HOST:5432/DATABASE
POSTGIS_URL=postgresql://USER:PASSWORD@HOST:5432/DATABASE

OPENAI_BASE_URL=
OPENAI_MODEL=gpt-4o-mini
OPENAI_API_KEY=

CORS_ORIGINS=http://localhost:3000,http://localhost:5173
```

در محیط Production، نمونه تنظیمات دامنه‌ها:

```env
VITE_API_URL=https://api.mapathon.ir
NEXT_PUBLIC_API_URL=https://api.mapathon.ir
CORS_ORIGINS=https://mapathon.ir,https://www.mapathon.ir
```

> فایل `.env` نباید در Git commit شود. حتماً آن را در `.gitignore` قرار دهید.

## اجرای پروژه با Docker

برای اجرای سرویس‌ها با Docker Compose:

```bash
docker compose up -d --build
```

مشاهده لاگ سرویس‌ها:

```bash
docker compose logs -f
```

خاموش کردن سرویس‌ها:

```bash
docker compose down
```

ساخت مجدد سرویس‌ها:

```bash
docker compose build --no-cache
docker compose up -d
```

## پایگاه داده

مپاتون برای ذخیره و پردازش داده‌های مکانی از PostgreSQL و PostGIS استفاده می‌کند.

PostGIS امکان انجام عملیات زیر را فراهم می‌کند:

- ذخیره‌سازی داده‌های نقطه‌ای، خطی و چندضلعی
- جست‌وجوی مکانی
- محاسبه فاصله
- بررسی محدوده‌های جغرافیایی
- تحلیل ارتباط میان عوارض مکانی
- اجرای کوئری‌های مکانی پیشرفته

نمونه فعال‌سازی PostGIS:

```sql
CREATE EXTENSION IF NOT EXISTS postgis;
```
